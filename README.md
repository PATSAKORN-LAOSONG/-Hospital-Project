# Hospital-Project
ระบบการโทรเยี่ยมติดตามผู้ป่วย

## 💻 วิธีการทำงานขอระบบ
1) บันทึกข้อมูลผู้ป่วย
   - ชื่อ
   - เบอร์โทร
   - โรคหรืออาการ
   - วันที่เข้ารักษา
2) ระบบแจ้งเตือนเจ้าหน้าที่
   - เมื่อถึงเวลาที่ต้องติดตาม ระบบจะแจ้งเตือนให้โทรหาผู้ป่วย
3) การโทรติดตามอาการ
   - ระบบจะมีปุ่มให้กดโทรไปยังเบอร์ของผู้ป่วยและสอบถามอาการของผู้ป่วย
4) บันทึกผลการติดตาม
   - วันนี้อาการดีขึ้นไหม
   - รับประทานยาครบตามที่แพทย์แนะนำหรือไม่
   - มีอาการผิดปกติไหม

## 🔍 แนวทางออกแบบ UI
1) หน้า Dashboard
   - จำนวนผู้ป่วยทั้งหมด
   - ผู้ป่วยที่ต้องโทรติดตามในวันนี้
   - ผู้ป่วยที่ติดตามแล้ว
   - ผู้ป่วยที่ต้องติดตามซ้ำ (อาจจะเอาไปรวมกับผู้ป่วยที่ต้องโทรติดตามวันนี้)

### ตัวอย่าง UI
ระบบติดตามผู้ป่วย
| ผู้ป่วยทั้งหมด | ผู้ป่วยต้องติดตามวันนี้ |
| :---: | :---: |
| 120 | 15 |

---

ตารางรายชื่อผู้ป่วยที่ต้องติดตามวันนี้
| ชื่อ | โรค | วันที่รักษา | สถานะ(รอติดตาม,ติดตามแล้ว) | ปุ่มโทร |
| :--- | :--- | :---: | :--- | :--- |
| สมชาย | เบาหวาน | 10/03 | รอติดตาม | โทร |

2) หน้ารายชื่อผู้ป่วย
   - ช่องค้นหาผู้ป่วย
   - ตารางข้อมูล
   - ปุ่มดูรายละเอียด
